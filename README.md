# Software Engineering Challenges App

## Description
This is a simple Android app developed using Kotlin that features two activities. The main activity displays the user's full name and student ID, along with buttons to navigate to a second activity. The second activity lists common software engineering challenges.

## Features
- Main Activity: Displays name and student ID, with buttons for navigation.
- Second Activity: Lists software engineering challenges with a button to return to the main activity.

## Challenges Listed
1. Managing Requirements
2. Ensuring Software Quality
3. Handling Project Scope
4. Time Management
5. Team Communication

## Testing Information
- Device: Pixel 4 Emulator
- Android OS Version: 11 (API 30)
